HTML

(Voltar)

HTML (Linguagem de Marcação de HiperTexto) é o bloco de construção mais básico da web. Define o significado e a estrutura do conteúdo da web. Outras tecnologias além do HTML geralmente são usadas para descrever a aparência/apresentação (CSS) ou a funcionalidade/comportamento (JavaScript) de uma página da web.

"Hipertexto" refere-se aos links que conectam páginas da Web entre si, seja dentro de um único site ou entre sites. Links são um aspecto fundamental da web. Ao carregar conteúdo na Internet e vinculá-lo a páginas criadas por outras pessoas, você se torna um participante ativo na world wide web.

TAGS

(Voltar)

Tags HTML são instruções que informam ao navegador como formatar o texto. Você pode usar tags para definir itálico, quebras de linha, objetos, bullet points e muito mais. Essas tags estão presentes no HTML (Hypertext Markup Language) de todas as páginas da internet

Os servidores leem o código HTML para entender e renderizar o conteúdo corretamente. Essa leitura é feita de cima para baixo, da mesma forma que você está lendo este artigo agora. Você pode usar quantas tags quiser para formatar o conteúdo. No entanto, existem algumas regras essenciais sobre tags HTML que você precisará seguir. Uma tag HTML deve conter três partes: Uma tag de abertura — começando com um símbolo < >; Conteúdo — instruções curtas para exibir o elemento na página; Uma tag de fechamento — terminando com um símbolo . Algumas tags HTML podem ser abertas. Isso significa que elas não precisam ser fechadas com . Essas tags são normalmente usadas para metadados ou quebras de linha.

Atributos em Tags

(Voltar)

Atributos de tags são informações adicionais que definem o comportamento de uma tag HTML. Eles são usados para identificar, estilizar, ou fornecer metadados sobre um elemento.

Tipos de atributos

  • Globais: Funcionam em todas as tags, como class, id, lang e style
  • Específicos: São direcionados para cada tag, como src, disabled, href e label

Conclusão

(Voltar)

HTML, ou HyperText Markup Language, é a base da web como a conhecemos. É a linguagem que define a estrutura e o conteúdo de uma página da web. Sem HTML, não teríamos sites, blogs, aplicativos da web ou qualquer outra coisa que você possa imaginar online!

Imagine o HTML como um esqueleto que dá forma a uma página da web. Ele define os diferentes elementos, como cabeçalhos, parágrafos, imagens, tabelas e muito mais. Cada elemento tem um papel específico para desempenhar na organização e apresentação do conteúdo.

O HTML é uma linguagem relativamente simples de aprender, especialmente se você tiver alguma familiaridade com outras linguagens de programação. Ele usa tags para definir os elementos, que são como instruções para o navegador sobre como exibir o conteúdo. Por exemplo, a tag

define um parágrafo, a tag h1 define um cabeçalho de nível 1, e a tag img insere uma imagem.

Apesar de sua simplicidade, o HTML é uma linguagem poderosa que permite criar páginas da web complexas e interativas. Ele pode ser usado para criar sites estáticos e dinâmicos, aplicativos da web, jogos e muito mais.

Mas o HTML não é tudo! Ele trabalha em conjunto com outras tecnologias, como CSS para estilos e JavaScript para interatividade, para criar experiências web completas e envolventes.
